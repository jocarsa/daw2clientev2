A partir del ejemplo de creación:
	- Añade un botón para marcar tareas como completadas
		- Las tareas guardarán un atributo "data-estado" que 			puede ser pendiente o completada
		- El botón cambiará ese estado
		- En función del estado se cambiará un texto en la tarea 		y el css de esta
	- Añade un segundo botón para poder editar el texto de la tarea

Ampliación:
	- Añade title a los botones
	- Añade una imagen a la tarea
		- Las tareas pendientes tienen una imagen y las 			completadas otra
	- Añade un enlace a la tarea
		- El link de la web se guarda en un atributo
